## 👋 Sobre Mim

</br>
🎓 Sou estudante de Análise e Desenvolvimento de Sistemas, motivado por transformar ideias em soluções web robustas e bem estruturadas. </br>
💻 Tenho como foco principal o desenvolvimento backend, utilizando Java com o ecossistema Spring para construir APIs escaláveis, seguras e de fácil manutenção. </br>
